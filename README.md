# Storyteller by SoundSensei

## Install

### Step 1.

```
  $ git clone https://github.com/HanmaDaiki/Hackathon-SoundSensei-ChattyAI
  $ cd Hackathon-SoundSensei-ChattyAI
```
  
### Step 2.
 
Replace .envexample with .env and paste your keys into .env

### Step 3.

```
$ npm run start
```
